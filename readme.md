
# GraphVault ⚡

**El generador automático de MOCs jerárquicos más rápido del mundo para Obsidian**  
→ 25 000 carpetas escaneadas en ~12 segundos · 100% Python · Zero plugins

## Lo que hace 
- Escanea cualquier vault o carpeta caótica en segundos  
- Crea un MOC perfecto por cada carpeta (padre ↑ · subcarpetas ↓ · archivos por tipo)  
- Archivos grandes (>2 MB) → proxy automático en `!_adjuntos`  
- Sincroniza borrados, checkpoint, lotes, barra de progreso  
- Reanuda donde lo dejaste · lock anti-doble ejecución

## Instalación en 30 segundos
```bash
# 1. Descarga
curl -L https://github.com/tuusuario/GraphVault/releases/latest/download/GraphVault.py -o GraphVault.py

# 2. Crea carpeta "input" al lado del script y mete tu vault
# 3. Ejecuta
python GraphVault.py
